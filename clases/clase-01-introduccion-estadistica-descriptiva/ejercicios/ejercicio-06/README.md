# Ejercicio 1.2.30

Un diagrama de Pareto es una variación de un histograma de datos categóricos producidos por un estudio de control de calidad. Cada categoría representa un tipo diferente de no conformidad del producto o problema de producción. Las categorías se ordenaron de modo que la categoría con la frecuencia más grande aparezca a la extrema izquierda, luego la categoría con la segunda frecuencia más grande, y así sucesivamente. Suponga que se obtiene la siguiente información sobre no conformidades en paquetes de circuito: 

- Componentes averiados: 126
- Componentes incorrectos: 210
- Soldadura insuficiente: 67
- Soldadura excesiva: 54
- Componente faltante: 131

Construya un diagrama de Pareto.
